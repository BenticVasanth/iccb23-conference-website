# ICCB 2023 Conference Website

A responsive conference website developed for the International Conference on Cancer Biology (ICCB 2023).

## Features
- Fully responsive layout
- Clean and accessible UI
- Cross-browser compatibility
- Optimized HTML & CSS structure
- Mobile-friendly navigation

## Tech Stack
- HTML5
- CSS3
- Bootstrap

## Purpose
Designed to provide conference information, schedules, and registration details
with a user-friendly and professional interface.
